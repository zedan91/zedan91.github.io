AZOBSS Shopee JSON Auto Extractor

Cara install:
1. Buka Chrome/Edge.
2. Pergi chrome://extensions
3. Enable Developer mode.
4. Klik Load unpacked.
5. Pilih folder ini: azobss-shopee-json-extension
6. Buka Shopee product link.
7. Extension akan auto download JSON.
8. Import JSON itu dalam Add Affiliate > Shopee JSON Auto Import.

Nota:
Browser biasa tidak benarkan website auto buka F12, auto paste Console, atau auto Enter atas sebab keselamatan. Extension ini ialah cara auto yang betul.
