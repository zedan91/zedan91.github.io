AZOBSS Shopee JSON Extractor - Install Chrome/Edge Extension

1. Buka Chrome/Edge.
2. Pergi ke: chrome://extensions atau edge://extensions
3. ON-kan Developer mode.
4. Klik Load unpacked.
5. Pilih folder: azobss-shopee-json-extension
6. Siap.

Cara guna:
1. Di Add Affiliate, paste link Shopee pada bahagian Shopee JSON Auto Import.
2. Tekan Accept & Open Shopee.
3. Shopee page akan terbuka.
4. Extension akan cuba auto download JSON ke folder Downloads/AZOBSS-Shopee-JSON.
5. Kembali ke website AZOBSS, tekan Import JSON dan pilih fail JSON itu.

Nota:
- Browser tidak membenarkan website auto buka F12/Console/paste script kerana security.
- Extension ini ganti fungsi F12: ia extract data terus dari page Shopee dan download JSON automatik.
- Jika Shopee masih block title, guna Product Title Backup / Auto Fill.
