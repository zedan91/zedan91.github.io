# PHP Backend

GitHub Pages tidak menjalankan PHP.

Gunakan hosting PHP jika mahu fungsi berikut:

- Semak kewujudan PA sebelum buka tab JUPEM.
- Convert fail TIF kepada PDF.
- Simpan live traffic dan total visit menggunakan fail JSON server.

Keperluan hosting:

- PHP.
- cURL.
- Imagick untuk TIF ke PDF.
- Permission menulis fail jika guna `traffic.php`.
